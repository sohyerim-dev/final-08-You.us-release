# 기능 정의서

## 공통 규칙

### 접근 및 권한

- 로그인 필요한 기능은 비로그인 시 로그인 페이지로 이동한다.
- 본인 정보(주문, 배송, 후기, 관심상품)는 본인만 조회 및 수정할 수 있다.

### 입력 및 처리

- 필수 입력 항목은 명확히 표시한다.
- 제출 시 입력값에 대한 유효성 검사를 수행한다.
- 중복 제출을 방지하기 위해 처리 중 버튼을 비활성화한다.

### 데이터 기준

- 화면에 노출되는 데이터는 서버 기준을 따른다.
- 가격, 수량, 할인 금액은 서버 기준 데이터를 사용한다.
- 서버 응답 데이터가 없는 경우에도 화면은 깨지지 않아야 한다.

### 상태 유지

- 자동 로그인 시 로그인 상태는 로그아웃 전까지 유지된다.
- 비 자동 로그인 시 로그인 상태는 브라우저를 닫을 때까지 유지된다.
- 

### 오류 및 피드백

- 주요 사용자 행동에 대해 성공 또는 실패 피드백을 제공한다.

- 오류 발생 시 사용자에게 안내 메시지를 제공한다.
